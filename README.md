## 功能介绍（完整见运行截图）
管理员：管理员可以通过登录和注册访问系统，具备管理用户信息、课程信息、设备信息及新增课程功能，同时能审核课程推荐和特价设置，查看及管理后台日志，便捷管理全局。  
用户：用户通过注册和登录功能进入系统后，可浏览系统; 课程分类明确，包括做饭课程、美食分享和厨艺技巧等；可查看厨师信息并进行课程预约，同时拥有个性化信息管理，如修改个人信息和查看历史预约，系统支持上传头像，便捷交互提升用户体验。
